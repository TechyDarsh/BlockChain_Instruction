Blockchain Tracking Instructions

1. Blockchain Tracking Overview

This project is a blockchain-based system designed to track chemicals from production to delivery, providing transparency and security.

2. Technologies Used

The following technologies are used:

  • Solidity: Ethereum Blockchain Smart Contracts
  • React.js: Frontend User Interface
3. Getting Started

Follow these steps to install, configure, and run the project.

Clone This Repository www.github.com/TechyDarsh/BlockChain.git

4. Prerequisites

Ensure you have the following software installed on your machine:

  • Ganache: Local blockchain for Ethereum development.
  • MetaMask: Chrome extension for managing Ethereum accounts.
  • Node.js: Required to run the React frontend.
5. Installation and Setup

Steps to install and set up the project:

  1. Download and install Ganache.
  2. Start Ganache to set up your local Ethereum blockchain environment.
  3. Add MetaMask extension to your browser.
  4. Connect MetaMask to Ganache by adding a custom RPC network.
  5. Import an account from Ganache into MetaMask using the private key from Ganache.
6. Deploying the Smart Contract

To compile and migrate the smart contract:

  1. Open the project folder in your IDE.
  2. Copy the Ethereum address from Ganache into the App.js file.
  3. In your terminal, navigate to the project root and run the following commands:
    4. truffle compile
    truffle migrate
  4. Copy the ABI and contract address from the generated YourContract.json file.
  5. Update Contract.js with the ABI and contract address.
7. Running the Frontend

To start the React application:

npm start

This will open the application in your default browser at http://localhost:3000/.

8. Using the Application

Steps to use the application:

  1. Login with the following credentials:
    2. Username: admin
    Password: admin
  2. Create a batch of chemicals by filling in the details and clicking 'Create Batch'.
  3. Verify the blockchain transaction by checking the new block in Ganache.
9. Role-Based Access Control

The application supports multiple user roles:

  • Admin: Manages the system and oversees operations.
  • Manufacturers: Can create chemical batches.
  • Buyers: Can view and track the chemicals.
  • Customers: Can view the chemical batches but with limited permissions.
10. Troubleshooting

If you encounter any issues, ensure that:

  • Ganache is running and connected to MetaMask.
  • The correct network is selected in MetaMask.
  • The ABI and contract addresses are properly updated in Contract.js.
